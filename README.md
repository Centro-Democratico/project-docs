### Descripción:

Software especializado en benchmarking diseñado para realizar un análisis del hardware del dispositivo del usuario, identificando y evaluando automáticamente componentes clave como CPU, GPU, memoria RAM y almacenamiento, con el objetivo de medir su rendimiento. La aplicación compara estos resultados con una base de datos actualizada de componentes disponibles en el mercado, posicionando el equipo dentro de un ranking según métricas estandarizadas de rendimiento. Además, integra una interfaz de monitorización en tiempo real que incluye conteo de fotogramas por segundo (FPS), permitiendo visualizar de forma clara y precisa el comportamiento del sistema durante pruebas y ejecuciones en entornos reales, facilitando así la toma de decisiones informadas tanto para usuarios domésticos como para profesionales que requieren evaluaciones técnicas.
